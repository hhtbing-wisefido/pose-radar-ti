## 🔄 推送到 GitHub

### 方法 1：使用脚本（推荐）

1. **编辑 `push_to_github.bat`**
   ```batch
   set GITHUB_REPO=https://github.com/您的用户名/仓库名.git
   ```

2. **运行脚本**
   ```bash
   push_to_github.bat
   ```

### 方法 2：手动命令

```bash
# 添加远程仓库
git remote add origin https://github.com/您的用户名/仓库名.git

# 推送
git push -u origin master
```
