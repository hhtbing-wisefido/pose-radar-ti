### 评估目标

评估 **AWRL6844** 是否能实现以下功能：
- 人存检测（Presence Detection）
- 姿态检测（Pose Detection）
- 跌倒检测（Fall Detection）
- 呼吸检测（Breathing Detection）
- 心率检测（Heart Rate Detection）
