3. **设置SOP开关** - 功能模式（稍后烧录时会切换）
   - S7: ON（功能模式）
   - S8: ON（功能模式）
4. **上电** - 按下电源开关

---

### 1.2.2 COM口识别

#### 查看COM端口

打开 **设备管理器** → **端口(COM和LPT)**

**应该看到3个端口**：

| 端口名称 | 用途 | 说明 |
|---------|------|------|
| XDS110 Class Application/User UART | CLI命令端口 | 波特率115200 |
| XDS110 Class Auxiliary Data Port | 数据输出端口 | 波特率1250000 |
| XDS110 Debug Probe | JTAG调试端口 | CCS调试用 |

**记录COM端口号**：
```
例如：
- CLI端口: COM3
- 数据端口: COM4
- 调试端口: COM5
```
