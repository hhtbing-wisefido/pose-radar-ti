## 🏠 实际安装配置

### 安装方式

**墙面壁挂安装**

```
                墙面
                │
                │  ╔═══════════╗
                │  ║ AWRL6844  ║  ← 雷达模块
                │  ║   4TX 4RX ║
                │  ╚═════╦═════╝
                │        ║
                │        ║ 天线朝向
          1.78m │        ║
    (离地高度)   │        ▼
                │
    ════════════┴═══════════════  ← 地面
                
           监测区域 (4m宽 × 5m深 × 2m高)
```

### 安装参数详细说明

#### 位置坐标

```properties
# 传感器在世界坐标系中的位置
X坐标: 0.00 m    (以雷达为中心，左负右正)
Y坐标: 0.00 m    (距离墙面的位置，墙面为0)
Z坐标: 1.78 m    (离地高度)

# 传感器朝向角度
方位角 (Azimuth):   0°    (正前方)
俯仰角 (Elevation): -40°  (向下倾斜40度，俯视角度)
```

#### 配置文件格式

```properties
% 在.cfg配置文件中的写法
sensorPosition 0 0 1.78 0 -40
% 格式：X Y Z azimuthTilt elevationTilt
```

**参数说明**:
- **X, Y, Z**: 传感器位置坐标（单位：米）
- **azimuthTilt**: 方位角偏移（单位：度，0为正前方）
- **elevationTilt**: 俯仰角偏移（单位：度，负数表示向下倾斜）

## 📐 监测区域定义

### 主检测区域（Zone 0）

```
俯视图 (Top View):
                Y轴 (深度)
                ↑
                │
         ───────┼───────  ← 墙面 (Y=0)
                │
     -2.0m      │      +2.0m
       ├────────┼────────┤ ← X轴范围 (宽度)
       │        │        │
       │        ○        │ ← 雷达位置 (0, 0, 1.78)
       │     监测区      │
       │        域       │
       └────────┴────────┘
            Y=0.2m → 5.0m

侧视图 (Side View):
    Z轴 (高度)
    ↑
    │
2.0m├──────────────┐
    │              │
1.78m  ← ○雷达    │  ← 监测高度范围
    │              │
0.0m├──────────────┘ ← 地面
    └──────────────→ Y轴
      0.2m     5.0m
```

### 区域参数表

| 轴向 | 最小值  | 最大值  | 范围   | 说明                 |
| ---- | ------- | ------- | ------ | -------------------- |
| X轴  | -2.0 m  | +2.0 m  | 4.0 m  | 宽度（左右范围）     |
| Y轴  | 0.2 m   | 5.0 m   | 4.8 m  | 深度（距离墙面）     |
| Z轴  | 0.0 m   | 2.0 m   | 2.0 m  | 高度（地面到2米）    |

**配置文件写法**:
```properties
% Zone 0 - 主检测区域
cuboidDef 0 0   -2.0   2.0   0.2   5.0   0.0   2.0
cuboidDef 0 1   -2.0   2.0   0.2   5.0   0.0   2.0
cuboidDef 0 2   -2.0   2.0   0.2   5.0   0.0   2.0
% 格式：zoneId subZoneId xMin xMax yMin yMax zMin zMax
```

## ⚠️ 重要注意事项

### 安装位置调整影响

**如果需要修改安装高度或角度**，必须相应调整以下参数：

1. **sensorPosition参数** - 传感器位置和朝向
2. **cuboidDef参数** - 检测区域范围
3. **杂波滤除参数** - 静态杂波（墙面、地面）的滤除区域

**示例**：如果改为2.0m高度安装
```properties
# 需要修改的参数
sensorPosition 0 0 2.0 0 -45      # Z坐标从1.78改为2.0，俯角可能需要调整
cuboidDef 0 0 -2.0 2.0 0.2 5.5 0.0 2.5  # 检测范围也需要相应调整
```

### 天线朝向

- ✅ **正确**: 天线面板朝向监测区域
- ❌ **错误**: 天线背向监测区域（检测不到目标）
- ⚠️ **注意**: AWRL6844的天线在PCB正面（有芯片的一面）

### 遮挡问题

**避免以下情况**:
- ❌ 雷达前方有金属物体（强反射）
- ❌ 雷达前方有玻璃/镜子（多次反射）
- ❌ 雷达安装在角落（墙角杂波）
- ❌ 监测区域内有大型金属家具（强杂波）

### 电源和连接

**推荐连接方式**:
```
[PC USB端口] ──USB线──> [AWRL6844 Micro-USB]
                         │
                         ├─> 供电 (5V)
                         └─> UART通信
```

**不推荐**:
- ❌ 使用手机充电器供电（可能电流不足）
- ❌ 使用USB Hub（可能通信不稳定）
