# 依赖安装规范

规则类型: 强制规则
强制级别: 严格执行

---

## 核心规则

### 遇到依赖安装慢或无响应时

必须立即执行(不等待超过30秒):

1. 优先使用SOCKS5代理:
   pip install --proxy socks5://127.0.0.1:4000 包名

2. 备选:使用国内镜像:
   pip install -i https://pypi.tuna.tsinghua.edu.cn/simple 包名

---

## 镜像源列表

清华: https://pypi.tuna.tsinghua.edu.cn/simple
阿里云: https://mirrors.aliyun.com/pypi/simple
腾讯云: https://mirrors.cloud.tencent.com/pypi/simple
华为云: https://repo.huaweicloud.com/repository/pypi/simple

---

## AI执行要求

1. pip安装超过30秒无响应 立即切换代理/镜像
2. 不等待 不询问 直接切换
3. 优先顺序: SOCKS5代理 > 清华镜像 > 其他镜像

---

## AI强制自检

在执行pip install之前,AI必须检查:
- 我用了代理吗?
- 或者用了镜像吗?
- 如果都没有 立即修改命令!

绝对禁止:
- 直接 pip install 包名
- 等待超过30秒不切换
- 问用户要不要换源

---

AI必须主动执行,不等待用户指示!
